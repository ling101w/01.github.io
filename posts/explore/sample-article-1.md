# 我的第一篇博客文章

欢迎来到我的博客！这是我的第一篇文章。

## 关于这个博客

这个博客系统具有以下特点：

- 📝 支持 Markdown 格式写作
- 🚀 自动从 Git 仓库读取文章
- 📱 响应式设计，支持移动端
- 🎨 现代化的 UI 设计

## 如何使用

1. 在 `posts` 文件夹中创建 `.md` 文件
2. 使用 Markdown 语法写作
3. 推送到 Git 仓库
4. 网站会自动显示新文章
